Las componentes de nuestra batería son:

  • Mid Tom
  • High Tom
  • Tom Base
  • Base drum
  • Snare drum
  • Hit Hat
  • Ride
  • Crash 1
  • Crash 2

Instrucciones para tocar la batería desde el teclado

Cada componente de la bateria tiene una tecla específica asignada:

Funcionamiento platillos

Presiona:
- A para el Hit Hat
- A para el Crash 1
- A para el Crash 2
- A para el Ride

Funcionamiento bombos

Presiona:
- A para el Mid Tom
- A para el High Tom
- A para el Tom Base
- A para el Bass Drum
- A para el Snare Drum
